EL PROGRAMA HECHO CON LA NUEVA BASE DE DATOS

Ya que todas las librerias estan instaladas en el entrono virtual lo unico que vamos a necesitar para ejecutar la app es activar el entorno virtual. La app esta hecha en windows por lo tanto el comando seria:

kivy_venv/Scripts/Activate (con esto deberia abrir la app sin problemas, ejecutando el main.py)


La base de Datos se encuentra en un servidor gratuito, los datos de conexion estan en el archivo conectar_db.py en la carpeta REMISPOINT/screens.

Para poder ejecutar el proyecto deberemos instalar:

Kivy
KivyMD
KivyGarden
Psycopg2
googlemaps
requests
